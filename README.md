## Installation

```shell
git clone https://github.com/jeremy-code/libxslt-wasm-demo.git
cd libxslt-wasm-demo
corepack enable
pnpm install
```

## Usage

```shell
# Run the development server locally
pnpm dev

# Lint and format code with ESLint and Prettier
pnpm lint
pnpm format

# Build
pnpm build
```
